# 🌸 Proyecto Final Python - Blog de Naty ✨

¡Bienvenido/a a mi Entrega Final del curso de Python en Coderhouse! Este es un Blog diseñado con **Django**, enfocado en una estética moderna y amigable, que cumple con todos los requisitos solicitados en la consigna.

## 📝 Descripción del Proyecto
La aplicación permite a los usuarios registrarse, crear perfiles con avatar, publicar artículos (blogs) con imágenes y texto enriquecido, y comunicarse entre sí mediante un sistema de mensajería interna.

### 🚀 Funcionalidades Principales
* **Gestión de Usuarios (Accounts):** Registro, Login, Logout y Edición de Perfil (nombre, email, biografía y avatar).
* **Blog (Pages):** Listado de artículos, detalle de cada post, y un sistema completo de CRUD (Crear, Editar y Borrar) para usuarios logueados.
* **Mensajería:** App interna para que los usuarios puedan enviarse mensajes entre ellos.
* **Texto Enriquecido:** Implementación de `django-ckeditor` para dar formato a los artículos.
* **Sobre Mí:** Vista dedicada con información del autor.

## 🛠️ Tecnologías Utilizadas
* **Python 3.10+**
* **Django 4.2+**
* **Bootstrap 5** (Para el diseño rosa y responsive)
* **SQLite** (Base de datos por defecto)
* **Pillow** (Gestión de imágenes)
* **CKEditor** (Texto enriquecido)
